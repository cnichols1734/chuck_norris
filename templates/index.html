Chuck Norris Jokes API

Introduction

Welcome to the Chuck Norris Jokes API. Chuck Norris doesn't use APIs; APIs use Chuck Norris. But for the rest of us mortals, this service provides random Chuck Norris jokes for your applications. No authentication required (Chuck Norris doesn't believe in passwords).

Warning: Chuck Norris once roundhouse kicked this API. It's been running scared ever since. Use responsibly.

Base URL

All API requests should be made to (if you dare):

http://whatevermypythonanywheredomainis.com

Endpoint

GET /jokes/random

Returns a random Chuck Norris joke. Brace yourself.

Example Request:

GET http://whatevermypythonanywheredomainis.com/jokes/random

Response Structure

The API returns a JSON object with the following structure (Chuck Norris approved):

  • id (integer): A unique identifier for the joke. Chuck Norris counted to infinity. Twice. This is how many jokes we have.
  • joke (string): The text of the Chuck Norris joke. May cause spontaneous roundhouse kicks.

Rate Limits

To ensure fair usage (and to prevent Chuck Norris from breaking the internet), the API is limited to 1 request per second (60 requests per minute) per IP address. Exceeding this limit will result in a 429 Too Many Requests response. Chuck Norris doesn't abide by rate limits, but you should.

Error Handling

The API uses conventional HTTP response codes to indicate success or failure of requests (Chuck Norris never fails, but your requests might):

  • 200 - OK: Request successful. Chuck Norris approves.
  • 404 - Not Found: No jokes found in the database. Chuck Norris is momentarily speechless.
  • 429 - Too Many Requests: Rate limit exceeded. Chuck Norris is not amused.
  • 500 - Internal Server Error: Something went wrong on our end. Chuck Norris is coming to fix it, personally.
